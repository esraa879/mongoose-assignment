# Mongoose Assignment 9

## Features
- User Signup (hashed password + encrypted phone)
- Login with JWT (expires in 1 hour)
- Protected routes with auth middleware
- User CRUD (except password update)
- Notes CRUD (linked to logged-in user only)
- Custom validator: title must not be entirely uppercase

## Tech Stack
- Node.js
- Express
- MongoDB
- Mongoose
- JWT
